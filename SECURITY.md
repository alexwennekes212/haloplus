# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Email security reports privately to **info@gethaloplus.com** with:

- A clear description of the issue.
- Steps to reproduce it.
- Any affected version or browser.
- The likely impact.

The project owner will review the report and follow up when possible.

## Supported Versions

Security fixes are generally made against the latest public version of HaloPlus.
