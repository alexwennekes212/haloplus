Getting started

How to use HaloPlus

  • Press / anywhere in Halo to open the command palette.
  • Use the Data Viewer to explore records, filter data, and copy SQL.
  • Use Utilities for quick actions like Ticket 360, recent records, JSON inspection, and field API names.
  • Use this Settings page to manage preferences and custom slash commands.

About HaloPlus

Built from real Halo friction.

HaloPlus started as a practical response to the little bits of friction that pile up in Halo every day: settings that take too many clicks to find, records that are hard to jump back to, API and reporting work that needs more visibility, and the constant feeling that power users should be able to move faster.

The extension grew feature by feature from that idea: make the product easier to navigate, easier to inspect, and easier to work with for admins, consultants, and agents who live in Halo all day.
