# Contributing to HaloPlus

Thanks for helping improve HaloPlus.

## Before You Start

Please open an issue before working on larger features or behavior changes. Small fixes, typo corrections, and narrow bug fixes can be opened directly as pull requests.

## Development Setup

1. Clone the repository.
2. Load the repository as an unpacked extension from `chrome://extensions`.
3. Open a supported Halo page.
4. Make your changes and refresh the extension.

## Pull Requests

Please keep pull requests focused and easy to review.

- Explain the problem being solved.
- Include screenshots or short notes for UI changes.
- Mention any tested Halo pages or flows.
- Avoid unrelated formatting or refactors in the same pull request.

## Project Boundaries

HaloPlus is public for inspection and contribution. Republishing the extension, modified versions, or substantial parts of its functionality to extension stores or other marketplaces is not allowed without prior written permission from the owner.
