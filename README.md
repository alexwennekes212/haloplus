# HaloPlus

HaloPlus is a browser extension that adds productivity, workflow, and data tools for HaloITSM and HaloPSA.

It is built for admins, consultants, and agents who spend a lot of time inside Halo and want faster navigation, better record visibility, SQL/reporting helpers, and practical in-page utilities.

## Features

- Command palette for quickly finding records and actions in Halo.
- Data Viewer for exploring Halo records and copying generated SQL.
- Schema browser for Halo tables and columns.
- Ticket 360 and action timeline utilities.
- Recent records, field API name overlays, and raw record JSON inspection.
- Custom slash commands stored locally in the browser.

## Supported Sites

HaloPlus currently runs on:

- `*.halopsa.com`
- `*.haloitsm.com`
- `*.haloservicedesk.com`

## Install for Development

1. Clone this repository.
2. Open Chrome or a Chromium-based browser.
3. Go to `chrome://extensions`.
4. Enable **Developer mode**.
5. Click **Load unpacked**.
6. Select this repository folder.
7. Open or refresh a supported Halo page.
