# Privacy Policy

HaloPlus is designed to run inside supported HaloITSM, HaloPSA, and Halo Service Desk pages.

## Data Access

The extension may read page context from supported Halo pages so it can provide in-page tools such as record navigation, field helpers, data viewing, JSON inspection, and SQL/reporting utilities.

## Local Storage

HaloPlus may store preferences, recent records, and custom slash commands in browser extension storage.

## External Services

HaloPlus does not intentionally send personal data, Halo data, or browsing data to external analytics services from this repository version.

## Browser Permissions

HaloPlus requests access only for supported Halo domains listed in `manifest.json`.

## Contact

Open an issue in this repository for privacy questions that do not involve sensitive information.
